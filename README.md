# Playground

Random stuff I've accumulated over time.

## What's here

- **[debian-docker-zfs/](debian-docker-zfs/)** - guide for setting up Debian Trixie with ZFS root and a Docker homelab platform on top
- **[scripts/](scripts/)** - utility scripts (video duration calculator, comment stripper, docker updater)
- **[notes/](notes/)** - quick notes on LVM, ZFS, stuff I keep forgetting
- **[vss-migration/](vss-migration/)** - tools for migrating old Visual SourceSafe repos to git
- **[docker-alpine-git/](docker-alpine-git/)** - minimal alpine container with git
- **[XY-WPCE-Firmware/](XY-WPCE-Firmware/)** - firmware backup for XY-WPCE device

Mostly things I've needed once or twice and figured I'd keep around.

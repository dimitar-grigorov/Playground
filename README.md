# Playground

Random stuff I've accumulated over time.

## What's here

- **scripts/** - utility scripts (video duration calculator, comment stripper, docker updater)
- **notes/** - quick notes on LVM, ZFS, stuff I keep forgetting
- **vss-migration/** - tools for migrating old Visual SourceSafe repos to git
- **docker-alpine-git/** - minimal alpine container with git
- **XY-WPCE-Firmware/** - firmware backup for XY-WPCE device

Mostly things I've needed once or twice and figured I'd keep around.
